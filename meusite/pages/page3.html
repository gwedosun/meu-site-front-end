Formatação de Texto

A tag <strong> é usada para dar importância forte a um texto; leitores de tela e buscadores reconhecem essa importância. Já a tag <b> apenas aplica o estilo negrito visualmente, sem adicionar peso semântico.

De forma similar, a tag <em> aplica uma ênfase ao texto, alterando a entonação da leitura. A tag <i> representa um texto que se diferencia do normal (como um termo técnico ou um pensamento), e que visualmente costuma ser em itálico.

Podemos usar a tag <mark> para destacar uma parte do texto, como se fosse um marca-texto.

Para indicar edições, usamos a tag <del> para texto removido e a tag <ins> para texto inserido.

A tag <small> é usada para textos com menor destaque, como notas de rodapé ou direitos autorais. Este é um exemplo de texto menor.

A tag <sub> é usada para criar texto subscrito, como na fórmula química da água: H2O.

A tag <sup> é usada para criar texto sobrescrito, útil para expoentes matemáticos, como em X2.
